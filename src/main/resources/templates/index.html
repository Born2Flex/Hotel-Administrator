Welcome, Administrator!

As an administrator, you have access to various functions to manage the hotel:

  • Rooms: View and the list of rooms.
  • Guests: View and manage guest information, including passport data and check-in/check-out dates.
  • Check-In: Check-in guests, assign rooms.
  • Check-Out: Check-out guests, free up rooms, and issue additional checks if needed.
  • Search: Search for guests using any criteria.

Use the navigation links above to access these functions.
